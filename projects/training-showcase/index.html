Excel (Power Query + PivotTables + Charts)

Goal: Quick KPIs, light modeling, printable executive readout.

Steps
  1. Open Portfolio_Starter.xlsx or import CSVs via Power Query.
  2. Clean region labels, handle nulls, and add Net Revenue field.
  3. Create a Date table (Year, Month, Year-Month) for consistent time analysis.
  4. Build PivotTables: monthly trends, YoY helper, category/subcategory revenue, channel conversion & ROI.
  5. Add charts: revenue trend lines, bar charts for categories and ROI, KPI cards.
  6. Add slicers for Year, Region, Channel, Category and a “Data Notes” sheet documenting cleaning steps.

Tableau

Goal: Interactive executive dashboard with KPIs, trends, and drilldowns.

Steps
  1. Connect to CSVs or Portfolio_Starter.xlsx; relate order_items→products, and use orders, web_sessions, channel_spend.
  2. Create calculated fields: Net Revenue, Month, YoY %, Conversion %, ROI.
  3. Sheets to build: KPI tiles; Monthly Net Revenue + YoY; Channels (Conversion %, ROI); Products (Top Categories); Regions (map or bars).
  4. Assemble a dashboard; add filters (Year, Region, Channel, Category) and descriptive tooltips.
  5. Publish to Tableau Public and link from your portfolio.

Power BI

Goal: Same visuals as Tableau with a DAX semantic model.

Steps
  1. Load CSVs; in Model view create relationships and a Date table:
    DateTable = CALENDAR(DATE(2023,1,1), DATE(2025,6,30))
  2. Add columns: Year, Month, YearMonth; write measures:
    Net Revenue, Orders, AOV, Return %, Sessions, Transactions, Conversion %, Revenue LY, YoY %, Spend, ROI.
  3. Create visuals: KPI cards; Revenue + YoY line; Conversion % by channel; ROI by channel; Revenue by category; Region momentum (recent vs prior 6m).
  4. Add slicers for Year, Region, Channel, Category; format and publish/export.

GitHub / Portfolio Site

Goal: Public, reproducible project with clear documentation.

Steps
  1. Repo structure:
    /data (CSVs), /analysis (SQL), /workbooks (Excel), /docs (summary/insights).
  2. README: READY overview, data schema diagram, reproduce steps, screenshots/GIFs of dashboards, key insights & recommendations.
  3. Commit, push, and (if using GitHub Pages) deploy the project page; link assets (dashboards, PDFs, downloads).

Word / Slide Deck

Goal: Concise narrative for executives (report + presentation).

Steps
  1. Word report: Executive summary; KPIs; trends; channel/product/region insights; recommendations; assumptions/limitations.
  2. Slide deck (6 slides): Title + KPIs; Trend & Seasonality; Channel Performance; Product Performance; Regional Growth; 90-Day Action Plan.
  3. Export to PDF and link from the project page.
